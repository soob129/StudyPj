# 스터디 구인 웹 사이트

## 프로젝트 소개
Spring Boot 기반의 스터디 모집 및 참여 관리를 위한 웹 사이트 개인 프로젝트 입니다.
게시글과 댓글을 중심으로 한 기본적인 커뮤니티 기능을 구현했으며,
관리자 관점에서 사이트를 관리할 수 있도록 설계한 것이 특징입니다.

## 기술 스택
- Backend: Java 17, Spring Boot
- Frontend: JavaScript(Fetch API), HTML/CSS
- ORM: JPA(Hibernate)
- Database: Oracle DB
- Build Tool: Maven
- Version Control: Git, GitHub

## 주요 기능
- 회원 관리(로그인, 권한 구분)
- 스터디 모집 게시글 CRUD
- 상태(Status) 컬럼을 활용한 게시글 / 댓글 논리 삭제 처리
- 관리자 권한을 통한 전체 회원 / 게시글 / 댓글 조회
- 관리자에 의한 게시글 / 댓글 상태 관리

## 구현 특징
- REST API 기반 서버 구축
- Fetch API를 활용한 비동기 통신 처리
- JPA 를 활용한 ORM 기반 데이터베이스 연동
- 데이터 삭제 시 상태값을 활용하여 데이터 이력을 유지하고 관리하도록 설계

## 한계 및 개선 방향
현재는 관리자가 직접 게시글과 댓글을 관리하는 구조로 구현되어 있습니다.
서비스 운영 측면에서 스팸성 게시글을 보다 효율적으로 관리하기 위해,
향후 사용자 신고 기능 도입 및 기준에 따른 자동 관리 기능으로 확장할 필요성을 인식했습니다.

## 느낀 점
단순히 웹사이트를 구현을 넘어,
직접 서비스를 운영하는 과정에서 필요한 관리자 기능과 
데이터 관리 방식에 대해 고민해볼 수 있는 프로젝트였습니다.

또한 개발을 진행하며 기능 요구가 점차 확장되는 경험을 통해,
초기 설계 단계에서 부터 추후 기능 확장을 고려하는 것이
중요하다는 점을 체감할 수 있었습니다.
